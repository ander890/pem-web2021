Tentang Saya

Hai saya Daniel Alexander dan saya adalah seorang Fullstack Developer. Saya mulai belajar pemrograman sejak SMP.

Pada saat saya pertama belajar adalah dengan studi kasus membuat toko online dengan PHP, HTML & CSS. Semenjak saat itu saya terus mengembangkan kemampuan saya sehingga menajdi fullstack developer.

Saat ini saya telah membuat berberapa aplikasi dan website seperti absensi online dengan menggunakan face recognition untuk salah satu rumah sakit di Ciamis. Saya juga membuat aplikasi marketing melalui whatsapp.

Saya biasa menggunakan PHP Laravel, Vue.js dan Java untuk membuat aplikasi.

Menurut saya coding merupakan hal yang menantang & seru, kita harus berfikir logis serta rasional untuk memecahkan masalah yang terjadi
